주소 프로세스
-> 받은 메시지 분류 가능해야 함(집으로만 들어오면 누구 것인지 모름)
-> 분류는 호스트 IP 주소와 포트 넘버
-> IP 주소:포트번호 형태 ex) 10.10.16.68:80
-> 포트 넘버는 겹쳐도 됨, 단 IP가 다를 때
-> 많은 호스트가 동일한 프로세스에서 실행 가능

전송/공유 서비스 요소 4가지(앱에 있어 필요한)
1. 데이터 무결성(data integrity)
2. 보안(security)
3. 시간(timing)
4. 대역폭, 통과 가능한 양 -> 효율성 (throughput)
